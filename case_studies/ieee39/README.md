\# IEEE-39 Case Study (Proof-of-Concept)



This case study demonstrates strong suppression of cascading failure

using logistic softening with memory feedback.



At 140 MW:

\- Baseline collapse probability ≈ 0.56

\- Controlled collapse probability ≈ 0.01

\- Δ ≈ 0.55



This represents a proof-of-concept before the emergence

of finite-width critical bands in larger systems.
